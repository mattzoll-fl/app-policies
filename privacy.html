Overview

This application is an internal tool used solely by authorized personnel for financial data management and automation.

This application accesses QuickBooks Online accounting data, including transactions, accounts, and related financial records, strictly for internal bookkeeping and reconciliation purposes.

Data accessed through this application is used exclusively for internal financial operations. It is not sold, shared, or disclosed to any third parties.

OAuth tokens and API credentials are stored locally on secured internal systems. No financial data is stored externally.
